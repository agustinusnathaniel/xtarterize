# Welcome to Xtarterize

Xtarterize is a toolkit designed to scaffold and configure full-stack TypeScript projects with best practices built-in.

## What is Xtarterize?

Xtarterize helps you quickly set up new projects with:

- **Project detection** - Automatically detect existing project configurations and frameworks
- **Smart patching** - Apply targeted modifications to configuration files without overwriting customizations
- **Task execution** - Run predefined tasks for common setup operations
- **CLI interface** - Simple command-line interface to scaffold new projects

## Quick Start

```bash
# Scaffold a new project
npx xtarterize init

# Or run from an existing project
npx xtarterize analyze
```
