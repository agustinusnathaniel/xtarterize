## Prerequisites

- [Node.js](https://nodejs.org/) 18 or later
- A [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)/[TypeScript](https://www.typescriptlang.org/) project (or starting fresh)

<Aside>
  xtarterize requires a [`package.json`](https://docs.npmjs.com/cli/v10/configuring-npm/package-json) with a `name` field and an initialized [Git](https://git-scm.com/) repository. These are validated by the preflight checks before any changes are made.
</Aside>

## Run Without Installing

You can use xtarterize without installing it globally by using `npx`:

```bash
npx xtarterize init
```

This is the recommended approach — you always get the latest version.

## Verify It Works

```bash
npx xtarterize --version
npx xtarterize --help
```
