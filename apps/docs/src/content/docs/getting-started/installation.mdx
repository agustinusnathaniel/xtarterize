# Installation

## Prerequisites

- Node.js 18 or later
- pnpm (recommended) or npm

## Install Xtarterize

You can use Xtarterize without installing it by using `npx`:

```bash
npx xtarterize init
```

Or install it as a development dependency:

```bash
# Using pnpm
pnpm add -D @xtarterize/cli

# Using npm
npm install --save-dev @xtarterize/cli
```

## Verify Installation

After installation, verify that Xtarterize is working:

```bash
npx xtarterize --version
```
